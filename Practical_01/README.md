Practical Title-
Addition of Two Time Objects using Operator Overloading

Description-
This program demonstrates operator overloading in C++.
A class named 'Time' is created with hours and minutes as data members. 
The '+" operator is overloaded to add two time objects and display the total time in hours and minutes.

Steps to Run/Execute
1. Open the program in any C++ compiler (VS Code / CodeBlocks / Dev C++).
2. Compile the program using a C++ compiler.
3. Run the executable file.
4. Enter hours and minutes for two time objects when prompted.
5. The program displays the sum of both times.
